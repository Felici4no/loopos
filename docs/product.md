# LoopOS — Produto

## O que é o LoopOS

LoopOS é um sistema pessoal modular para registrar e acompanhar rotina, corpo, leitura, hábitos e listas. É um app de uso diário, pessoal e offline-first, projetado para durar — não para surpreender.

A premissa central: a maioria dos apps de produtividade falha porque pede demais do usuário. O LoopOS parte do oposto: registrar o mínimo necessário para gerar clareza sobre como você vive.

---

## Módulos iniciais (v0.1)

### Hoje
Registro diário. O usuário entra, anota como foi o dia, define o humor do dia (1–5) e marca o dia como concluído. Simples e rápido.

### Corpo
Registro de treinos e atividade física. Tipo de treino, duração, nota livre. Sem gamificação. Só dado.

### Ritmo
Trackers personalizados de hábitos e frequências. O usuário cria um tracker (ex: "Meditação", "Água", "Cigarro") e registra eventos. Exibe streaks e histórico visual simples.

### Leitura
Biblioteca pessoal de livros. Status (quer ler, lendo, finalizado), sessões de leitura com páginas lidas, notas livres.

### Listas
Listas hierárquicas simples (nós com subnós). Sem projetos, sem datas, sem Kanban. Uma lista é só uma lista.

---

## Objetivo do MVP v0.1

**Provar que o usuário consegue usar o app por 7 dias consecutivos sem abandonar.**

Critérios de sucesso:
- O usuário abre o app todos os dias durante 7 dias.
- Registra pelo menos um dado por módulo ao longo da semana.
- Não encontra fricção técnica (crashes, lentidão, erros visíveis).

O que não é objetivo do v0.1:
- Beleza visual final.
- Recursos sociais ou compartilhamento.
- Análise inteligente ou IA.
- Widgets ou integrações externas.
